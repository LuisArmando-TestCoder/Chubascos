title: "dios-es-invisible"

Dios es invisible

para que sus amores

se vean

entre ellos.

&nbsp;

Dios se relegó

a ser una posibilidad

para no afectar a

las mariposas.

&nbsp;

Dios nos deja accidentarnos,

para que el Paraíso

llegue a su tiempo,

cuando estemos

cada uno

listos

para apreciarlo.
