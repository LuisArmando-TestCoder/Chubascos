title: "el-orgullo-se-pierde"
---
El orgullo se pierde,

pero no se pierde

el mecanismo que lo lleva,

no se pierde la nostalgia

que lo contenía,

no se olvida el hábito

de llevarlo colgando,

no se olvida el peso

de llevar la piedra

cuesta arriba

y roce de cadenas

que adornaban las llagas.
